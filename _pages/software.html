Below is a list of software I have developed, with links to the code repositories and related publications.

Software

  • phypanda
    GitHub: https://github.com/nholtgrefe/phypanda
    A Python package for maximization and exploration of phylogenetic diversity in phylogenetic networks.
    Paper: https://doi.org/10.1101/2025.11.14.688467

  • physquirrel
    GitHub: https://github.com/nholtgrefe/squirrel
    A Python package for reconstructing semi-directed phylogenetic level-1 networks from four-leaf subnetworks or sequence alignments.
    Paper: https://doi.org/10.1093/molbev/msaf067

  • ComputingScanwidth
    GitHub: https://github.com/nholtgrefe/ComputingScanwidth
    Algorithms and heuristics for computing the scanwidth of directed acyclic graphs.
    Paper: https://arxiv.org/abs/2403.12734
